# 图片资源说明

本目录用于存放网站所需的图片资源文件。

## 需要添加的图片

请将以下图片文件放置在本目录中：

### 1. 头像图片
- **文件名**: `avatar.jpg`
- **建议尺寸**: 400x400 像素
- **格式**: JPG 或 PNG
- **说明**: 个人头像照片，建议使用正方形图片，会以圆形方式展示

### 2. 学校校徽
- **文件名**: `school-logo.png`
- **建议尺寸**: 200x200 像素
- **格式**: PNG（支持透明背景）
- **说明**: 学校校徽图片，可选添加

### 3. 项目截图
- **文件名**: `project1.jpg`, `project2.jpg`, `project3.jpg`
- **建议尺寸**: 800x500 像素
- **格式**: JPG 或 PNG
- **说明**: 项目展示截图，建议使用清晰的项目界面截图

## 图片优化建议

1. **压缩图片**: 使用图片压缩工具（如 TinyPNG、ImageOptim）压缩图片，减少文件大小
2. **保持比例**: 保持图片比例一致，避免变形
3. **文件大小**: 单张图片建议不超过 500KB
4. **格式选择**:
   - 照片类使用 JPG 格式
   - 图标、Logo 类使用 PNG 格式（支持透明）
   - 简单图形可考虑使用 SVG 格式

## 图片版权

请确保使用的图片拥有合法的使用权，避免侵犯他人版权。
